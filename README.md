86230013 İrem Melisa Kılıçaslanoğlu Programlamaya Giriş Final Projesi

Bu proje, okuyucuların ruh haline göre ortam atmosferini değiştirebileceği, interaktif bir web uygulamasıdır. Kullanıcılar klavye tuşlarını kullanarak farklı senaryolar (Normal, Korku, Romantik) arasında kesintisiz geçiş yapabilirler.

Projeyi kendi bilgisayarınızda çalıştırmak için şu adımları izleyin:

1. Proje dosyalarını içeren klasörü veya ZIP dosyasını bilgisayarınıza indirin.
2. Klasör içerisindeki index.html dosyasının ve video dosyalarının (1.mp4, 2.mp4, 3.mp4) aynı yerde olduğundan emin olun.
3. index.html dosyasına çift tıklayarak veya dosyayı Google Chrome / Safari üzerine sürükleyerek tarayıcıda açın. 
4. Ekranda beliren "Başlat” butonuna tıklayarak deneyimi başlatın ve klavyedeki 1-2-3 tuşlarını kullanarak atmosferler arası geçiş yapın.
